Travel List Web App

About

I was tasked with creating a single-page web application using React to help travelers keep track of their packed items during their holidays.

Goals

My goal was to develop a minimalist, user-friendly application that simplifies the packing process for users.

Process

I began by adding form elements, including a quantity dropdown, a text input field, and an add button. Each of these elements was connected to useState hooks with event handlers to ensure state updates.

Next, I displayed the user-added items in a list format using the .map() function. Within this list, I included a checkbox to mark items as packed and a delete button (X) for removing individual items. A conditional style was applied to visually indicate packed items by striking through their descriptions.

I implemented a "delete all" function to clear the list entirely and created a status feature that calculates the percentage of items packed using a simple formula. A ternary operator was used to toggle between personalized messages when all items are packed.

For deployment, I committed my code to GitHub regularly and used Vercel to deploy my application for the first time.

Challenges

One challenge I faced was properly styling the packed items dynamically. Initially, I struggled to get the textDecoration: "line-through" style to apply only when the checkbox was ticked. By carefully reviewing how state updates work and ensuring the packed property of each item was toggled correctly, I managed to fix this issue.

Another challenge I faced was deploying the app for the first time using Vercel. Watching tutorials and experimenting with the deployment process helped me successfully publish the application.

Results

The final product is a fully functional travel list web app, deployed for users to track their packing efficiently and enjoy stress-free travel.
