Finance Manager Application

About

For this project, I developed a prototype mobile application using React Native, titled Finance Manager App. The app focuses on helping users manage their income and expenses, promoting better financial habits. With the rising cost of living, effective money management is more important than ever, and this app aims to empower users to track their finances and make informed decisions.

Goals

The app’s primary goal was to be user-friendly, intuitive, and to provide insights into spending habits to encourage smarter financial decisions.

Process

To start, I designed a low-fidelity prototype using Figma to map out the app’s layout and flow. This gave me a clear vision of the design direction. For the app’s structure, I used a SectionList component in React Native, as it accommodates multiple datasets and simplifies data retrieval during development.

Challenges

One of the challenges I faced was designing an interface that balances functionality and simplicity while keeping the app visually appealing. Additionally, learning how to efficiently implement the SectionList feature in React Native required extra research and troubleshooting. Everything went well as I was able to deliver the final prototype on time.

Results

The results was a fully functional prototype named Budget App. In the app, users can add, edit, and delete income and expense entries directly. The design features a clean, intuitive interface with sections for different expense categories, providing a clear breakdown of spending. I chose a color scheme of purple and brown to create a warm and engaging visual theme.

This project honed my skills in design and development, allowing me to create a practical tool that addresses an everyday challenge while delivering a seamless user experience.
